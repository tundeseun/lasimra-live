# LASIMRA Website Redesign

This project is a React + TypeScript + Vite rebuild of the public LASIMRA website structure and main public-facing content.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Notes
- Main styling is in `src/styles/main.css`
- Routing is handled with React Router
- Public logo asset is in `public/lasimralogo.png`
- Contact form is UI-only and not yet connected to a backend
